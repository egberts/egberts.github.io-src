Thunderbird Email Setup

The information that you will need is:

* Email Address:

* Account Username:

* Account Password:

* POP3 Incoming Mail Server: (mail.yourdomain.xxx)

* SMTP Outgoing Mail Server: (mail.yourdomain.xxx)

Other Settings

Outgoing Mail Server Requires Password Authentication (needs a password to send)

Change outgoing SMTP server port from 25 to 366 (some ISPs block port 25)

Setup Directions

* Click on Tools / Account Settings

* Find Account Actions button in Lower Left / Add Mail Account

* Enter Your Name / Email Address / Password / Check Remember Password / Create Account

* Enter Username / Enter Incoming Mail Server (mail.yourdomain.com) / Leave as POP3 / Leave Port as 110 / None for Security

* Outgoing Server Should be mail.yourdomain.com / Change SMTP Port from 25 to 366 / None for Security

* Create Account
